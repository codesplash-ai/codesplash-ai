## S FAQ

<details>
<summary><strong>What is CodeSplash AI?</strong></summary>

CodeSplash AI is a desktop application that lets you build full-stack applications using natural language prompts. It combines AI-powered code generation with live preview, version control, and one-click deployment.
</details>

<details>
<summary><strong>How does pricing work?</strong></summary>

CodeSplash is **free forever in local mode**just bring your own API keys from OpenAI, Anthropic, or other providers. Upgrade to Pro ($25/mo) or Max ($50/mo) for cloud features like collaboration, managed AI gateway, and team workspaces.
</details>

<details>
<summary><strong>Can I use my own API keys?</strong></summary>

Yes! In local mode (free forever), you bring your own API keys from any supported provider. In cloud mode, we provide a managed AI gateway, but you can still use your own keys if preferred.
</details>

<details>
<summary><strong>Which LLM providers are supported?</strong></summary>

OpenAI (GPT-4, GPT-3.5), Anthropic (Claude), Google (Gemini), xAI (Grok), Azure OpenAI, Amazon Bedrock, OpenRouter, and any OpenAI-compatible provider. Switch between providers anytime.
</details>

<details>
<summary><strong>Does it work offline?</strong></summary>

Yes! Local mode works fully offline once installed (except for AI API calls, which require internet). Your code is stored locally in SQLite. Cloud mode requires internet for collaboration and cloud features.
</details>

<details>
<summary><strong>Is my code private?</strong></summary>

Absolutely. In local mode, everything stays on your machine. In cloud mode, your code is stored securely with encryption at rest and in transit. You own your codeexport anytime as ZIP or push to GitHub.
</details>

<details>
<summary><strong>Can I export my apps?</strong></summary>

Yes! Export as ZIP, push to GitHub, or deploy to Vercel with one click. No vendor lock-inyour code is yours to use anywhere.
</details>

<details>
<summary><strong>How do updates work?</strong></summary>

CodeSplash checks for updates automatically and prompts you to install new versions. Updates are downloaded from GitHub Releases and installed seamlessly.
</details>

<details>
<summary><strong>What frameworks and languages are supported?</strong></summary>

CodeSplash generates modern web applications using React, TypeScript, Tailwind CSS, and Node.js by default. The AI can work with other frameworks and languages based on your prompts.
</details>

<details>
<summary><strong>Can I collaborate with my team?</strong></summary>

Yes! Pro and Max plans include real-time collaboration with live cursors, comments, and shared workspaces. Perfect for pair programming with AI.
</details>
